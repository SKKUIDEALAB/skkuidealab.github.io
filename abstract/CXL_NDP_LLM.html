Reducing Data Transfer Overhead with CXL-Based Near-Data Processing for LLM Inference

Woo Hyun Kim, Han Sol Kim, Tae Hee Han

15 October 2025

Abstract

In Large Language Models (LLMs), the Key-Value (KV) cache demands a substantial memory footprint as the input sequence length grows. Such memory requirements can exceed the size of model parameters and become unmanageable within limited GPU memory. In this paper, we offload the KV cache to Compute Express Link (CXL) memory and leverage Near-Data Processing (NDP) to select and transfer only the KV cache entries that highly affect model accuracy. Our approach reduces the data transfer overhead caused by limited CXL bandwidth and achieves an average 1.45× speedup over the baseline GPU with CXL memory.
